Project Description

For this assignment I rebuilt the homepage and FAQ page of the Mysteryland website using HTML, CSS and JavaScript. The goal was to reconstruct an existing website as accurately as possible, with attention to layout, styling and interaction.

Unlike other projects, I didn't go through an extensive design process here, such as making sketches or prototypes. This is because the design was already fixed and came from the original website. Instead, I analysed the existing site and translated it into code step by step.

During the process I mainly focused on understanding the structure of the website and rebuilding various components, such as the header with video, the navigation menu, interactive elements and the FAQ section. I used HTML for the structure, CSS for the styling and JavaScript for the interactions.
